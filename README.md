### blog-with-comments
# Run Locally

Clone the project

```bash
  git clone https://github.com/bogdanblare/blog-with-comment.git
```

Go to the project directory

```bash
  cd blog-with-comment
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
